# Age Calculator – Smart Date Engine

A premium-designed Age Calculator that demonstrates strong fundamentals in JavaScript date manipulation, validation logic, and UI design.

## Project Overview

This application calculates:

- Exact age (Years, Months, Days)
- Total days lived
- Total weeks lived
- Total hours lived
- Zodiac sign
- Time remaining until next birthday

It includes full Gregorian leap year logic and dynamic month-day adjustment to prevent invalid date selections.

## Engineering Highlights

- Correct leap year implementation:
  - Divisible by 4
  - Not divisible by 100
  - Unless divisible by 400
- Dynamic February handling (28/29 days)
- Future date prevention
- UI state management (no stale result rendering)
- Birthday detection logic with correct edge case handling

## Technologies

- HTML5
- CSS3 (Custom UI styling)
- JavaScript (Date object, DOM manipulation)
